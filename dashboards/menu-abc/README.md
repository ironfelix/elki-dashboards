# Menu ABC Analysis Dashboard

HTML report for management decisions based on a real iiko ABC export.

Included:
- point tabs (one tab per location sheet)
- period switcher
- actionable decisions (`protect`, `rework`, `test`, `remove`)
- no-sales detection vs previous period
- filters and search

Data source file in this folder:
- `abc_data.json` (generated from `/Users/ivanilin/Downloads/ABC.xlsx`)

Open locally:
- `dashboards/menu-abc/index.html`
